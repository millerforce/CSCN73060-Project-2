◆ getAcceptThread()

std::thread & ServerController::getAcceptThread ( )

Main loop for accepting incoming client connections.

Continuously listens for new client connections while the server is running. For each accepted connection:

  • Logs the connection event
  • Spawns a new thread to handle the client

◆ start()

void ServerController::start ( )

Starts the server controller.

Initializes the server loop and spawns a dedicated thread to accept incoming client connections.

This function is non-blocking; the accept loop runs in a separate thread.

◆ stop()

void ServerController::stop ( )

Stops the server controller.

Signals the server to stop accepting new connections, closes the server socket, and joins all running threads (accept thread and client threads).

This function blocks until all threads have finished execution.
